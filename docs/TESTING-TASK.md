# PageWise v2.1.0 — 全面功能测试任务

## 背景
PageWise 是一个 Chrome 扩展，用于页面内容理解、AI 对话、知识管理。当前有 1814 个测试用例，27 个失败。需要进行全面的功能测试和可靠性测试。

## 任务目标

### 1. 修复现有失败测试
- 运行 `node --test tests/test-*.js`
- 分析并修复 27 个失败用例
- 确保所有测试通过

### 2. 补充端到端测试
为以下模块编写完整的端到端测试：

**模块 1: 核心 AI 对话**
- 发送消息并接收 AI 回复
- 流式响应显示
- Markdown 渲染（代码块、表格、列表）
- 代码语法高亮
- 错误处理（API 错误、超时、网络断开）
- 多 Provider 切换（Anthropic/OpenAI/DeepSeek/本地）
- 对话历史持久化（IndexedDB）

**模块 7: 技能系统**

### 3. 可靠性测试
测试以下场景：
- 错误恢复能力（API 错误、网络断开、存储满）
- 边界条件（空输入、超长文本、特殊字符）
- 并发操作（同时保存/删除、多标签页操作）
