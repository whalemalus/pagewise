# BookmarkGraph 需求文档

> 基于 PRD.md 整理，融入 PageWise 项目架构
> 最后更新: 2026-05-03

---

## 产品定位

**一句话**：为程序员打造的收藏夹知识图谱工具，让收藏从"堆积"变成"可探索的知识网络"。

**与 PageWise 的关系**：BookmarkGraph 作为 PageWise 的新功能模块，复用现有知识图谱、语义搜索、学习路径等基础设施，扩展"书签维度"的知识管理能力。

---

## P0 — MVP 核心功能

### ✅ R043: 书签采集器 (BookmarkCollector)
- **描述**: 读取 Chrome 书签树，提取完整数据（标题、URL、文件夹路径、添加时间）
- **依赖**: `chrome.bookmarks` API（需新增 `bookmarks` 权限）
- **验收标准**:
  - 能递归读取完整书签树
  - 输出标准化书签对象数组
  - 处理空书签、重复书签、特殊字符标题
  - 支持 1000+ 书签，采集时间 < 5秒
- **复杂度**: Medium

### 📋 R044: 书签索引器 (BookmarkIndexer)
- **描述**: 对书签建立倒排索引，支持快速搜索
- **依赖**: R043
- **验收标准**:
  - 基于标题+URL+文件夹路径建立索引
  - 支持中英文分词
  - 搜索响应 < 100ms（1000条数据）
- **复杂度**: Medium

### 📋 R045: 书签图谱引擎 (BookmarkGraphEngine)
- **描述**: 基于关键词相似度构建书签关联网络
- **依赖**: R043, R044
- **验收标准**:
  - 提取书签关键词（标题分词 + URL域名 + 文件夹标签）
  - 计算书签间相似度（Jaccard / TF-IDF）
  - 生成图谱数据结构 `{nodes, edges}`
  - 相似度阈值可配置
- **复杂度**: Complex

### 📋 R046: 图谱可视化 (BookmarkVisualization)
- **描述**: 力导向图渲染，支持缩放/拖拽/搜索高亮
- **依赖**: R045
- **验收标准**:
  - Canvas 或 SVG 力导向图渲染
  - 支持鼠标缩放、拖拽平移
  - 节点颜色按文件夹/主题分类
  - 搜索时高亮匹配节点
  - 1000节点渲染流畅（>30fps）
- **复杂度**: Complex

### 📋 R047: 详情面板 (BookmarkDetailPanel)
- **描述**: 点击节点显示书签详情
- **依赖**: R046
- **验收标准**:
  - 显示：标题、URL、文件夹路径、添加时间
  - 显示：相似书签列表（基于图谱）
  - 点击URL可打开原网页
  - 支持编辑标签
- **复杂度**: Medium

### 📋 R048: 相似推荐 (BookmarkRecommendations)
- **描述**: 基于图谱推荐相似书签
- **依赖**: R045
- **验收标准**:
  - 选中一个书签，推荐 Top-5 相似书签
  - 显示相似度分数
  - 推荐结果可点击跳转
- **复杂度**: Medium

### 📋 R049: 书签搜索 (BookmarkSearch)
- **描述**: 在图谱中搜索书签
- **依赖**: R044, R046
- **验收标准**:
  - 搜索框实时过滤
  - 搜索结果高亮显示在图谱中
  - 支持按文件夹/标签过滤
- **复杂度**: Simple

### 📋 R050: 弹窗概览 (BookmarkPopup)
- **描述**: 点击扩展图标显示书签概览
- **依赖**: R043, R045
- **验收标准**:
  - 显示：书签总数、领域分布、最近添加
  - 快速搜索入口
  - "查看完整图谱"按钮跳转选项页
- **复杂度**: Medium

### 📋 R051: 选项页集成 (BookmarkOptionsPage)
- **描述**: 在选项页中集成完整图谱视图
- **依赖**: R046, R047, R048, R049
- **验收标准**:
  - 新增"书签图谱"标签页
  - 完整图谱 + 搜索 + 详情面板
  - 与现有标签页（问答/页面/技能/知识库/设置）风格一致
- **复杂度**: Medium

### 📋 R052: BookmarkGraph MVP 测试
- **描述**: MVP 阶段所有模块的 E2E 测试
- **依赖**: R043-R051
- **验收标准**:
  - 每个模块至少 10 个测试用例
  - 测试覆盖核心路径 + 边界情况
  - 所有测试通过
- **复杂度**: Medium

## P1 — V1.0 增强功能

### 📋 R053: 主题聚类 (TopicClustering)
- **描述**: 自动识别书签所属技术领域
- **依赖**: R045
- **验收标准**:
  - 基于关键词/URL模式自动分类（前端/后端/DevOps/AI等）
  - 支持 15+ 技术领域
  - 聚类结果可手动调整
- **复杂度**: Complex

### 📋 R054: 学习路径推荐 (LearningPathFromBookmarks)
- **描述**: 基于主题依赖推荐学习顺序
- **依赖**: R053, `lib/learning-path.js`
- **验收标准**:
  - 分析书签内容难度（入门/进阶/高级）
  - 生成学习路径：基础 → 实战 → 框架对比 → 生产实践
  - 标记已学/待学状态
- **复杂度**: Complex

### 📋 R055: 标签自动生成 (AutoTagGeneration)
- **描述**: 从书签内容自动生成标签
- **依赖**: R043
- **验收标准**:
  - 基于标题/URL/文件夹生成标签
  - 每个书签 3-5 个标签
  - 标签可去重、合并
- **复杂度**: Medium

### 📋 R056: 标签手动编辑 (TagManualEditing)
- **描述**: 用户可手动编辑书签标签
- **依赖**: R055
- **验收标准**:
  - 添加/删除标签
  - 标签自动补全
  - 批量编辑标签
- **复杂度**: Simple

### 📋 R057: 知识盲区检测 (KnowledgeGapDetection)
- **描述**: 发现收藏稀疏的主题领域
- **依赖**: R053
- **验收标准**:
  - 分析各领域书签数量
  - 识别"热门但资料少"的领域
  - 推荐补充资料的方向
- **复杂度**: Medium

### 📋 R058: 状态标记 (BookmarkStatusMarking)
- **描述**: 已读/待读/在读状态标记
- **依赖**: R043
- **验收标准**:
  - 三种状态：已读/待读/在读
  - 状态可批量修改
  - 按状态过滤书签
- **complexity**: Simple

### 📋 R059: 文件夹分析 (FolderAnalysis)
- **描述**: 分析书签文件夹结构
- **依赖**: R043
- **验收标准**:
  - 统计各文件夹书签数量
  - 识别"临时"等低质量文件夹
  - 建议整理方案
- **复杂度**: Simple

### 📋 R060: 重复检测 (DuplicateDetection)
- **描述**: 发现相似/重复书签
- **依赖**: R045
- **验收标准**:
  - 基于URL去重
  - 基于标题相似度去重
  - 合并/删除建议
- **复杂度**: Medium

### 📋 R061: 数据导入导出 (BookmarkImportExport)
- **描述**: 导出图谱 JSON/CSV
- **依赖**: R045
- **验收标准**:
  - 导出为 JSON（完整图谱数据）
  - 导出为 CSV（书签列表）
  - 导入 Chrome 书签 HTML 文件
- **复杂度**: Medium

### 📋 R062: BookmarkGraph V1.0 测试
- **描述**: V1.0 阶段所有模块的 E2E 测试
- **依赖**: R053-R061
- **验收标准**:
  - 每个模块至少 8 个测试用例
  - 集成测试覆盖模块间交互
  - 所有测试通过
- **复杂度**: Medium

## P2 — V2.0 高级功能

### 📋 R063: 链接健康检查 (LinkHealthCheck)
- **描述**: 检测书签链接是否失效
- **依赖**: R043
- **验收标准**:
  - 后台批量检测链接状态
  - 标记失效链接
  - 提供修复/删除建议
- **复杂度**: Medium

### 📋 R064: 书签内容预览 (BookmarkContentPreview)
- **描述**: 预览书签页面内容
- **依赖**: R043
- **验收标准**:
  - 提取页面标题+摘要
  - 显示页面截图（可选）
  - 缓存预览数据
- **复杂度**: Complex

### 📋 R065: 语义搜索 (BookmarkSemanticSearch)
- **描述**: 使用嵌入引擎进行语义搜索
- **依赖**: R044, `lib/embedding-engine.js`
- **验收标准**:
  - 支持自然语言查询
  - 语义相似度排序
  - 搜索延迟 < 500ms
- **复杂度**: Medium

### 📋 R066: 知识关联 (BookmarkKnowledgeCorrelation)
- **描述**: 将书签与现有知识库关联
- **依赖**: R045, `lib/knowledge-base.js`
- **验收标准**:
  - 书签与 Q&A 记录关联
  - 双向跳转
  - 关联强度可视化
- **复杂度**: Complex

### 📋 R067: 学习进度追踪 (BookmarkLearningProgress)
- **描述**: 追踪书签学习进度
- **依赖**: R058, R054
- **验收标准**:
  - 记录学习时间
  - 进度百分比
  - 学习统计图表
- **复杂度**: Medium

### 📋 R068: AI 推荐 (BookmarkAIRecommendations)
- **描述**: 基于 AI 的智能推荐
- **依赖**: R045, `lib/ai-client.js`
- **验收标准**:
  - 分析用户收藏模式
  - 推荐相关技术领域资料
  - 推荐理由说明
- **复杂度**: Complex

### 📋 R069: 统计仪表盘 (BookmarkStatistics)
- **描述**: 书签使用统计
- **依赖**: R043
- **验收标准**:
  - 收藏趋势图
  - 领域分布饼图
  - 活跃度热力图
- **复杂度**: Medium

### 📋 R070: 暗色主题 (BookmarkDarkTheme)
- **描述**: 图谱暗色模式
- **依赖**: R046
- **验收标准**:
  - 跟随系统/手动切换
  - 图谱节点/边颜色适配
  - 所有面板暗色适配
- **复杂度**: Simple

### 📋 R071: 快捷键 (BookmarkKeyboardShortcuts)
- **描述**: 书签图谱快捷操作
- **依赖**: R046
- **验收标准**:
  - 搜索：Ctrl+F
  - 缩放：+/-/0
  - 刷新：F5
  - 快捷键可自定义
- **复杂度**: Simple

### 📋 R072: BookmarkGraph V2.0 测试
- **描述**: V2.0 阶段所有模块的 E2E 测试
- **依赖**: R063-R071
- **验收标准**:
  - 每个模块至少 8 个测试用例
  - 性能测试（1000+ 书签）
  - 所有测试通过
- **复杂度**: Medium
