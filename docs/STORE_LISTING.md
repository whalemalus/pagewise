## 详细描述

### 中文版

**智阅 PageWise** — 你的 AI 网页阅读助手

浏览技术文档、API 文档、GitHub 仓库时，遇到不懂的内容？选中文字，直接向 AI 提问！所有回答自动整理成结构化知识库，随时回顾。

**🚀 核心功能**

• **智能问答**：选中网页文字即可向 AI 提问，支持流式输出和多轮对话
• **知识库管理**：回答自动保存到本地知识库，支持全文搜索、标签分类、知识图谱可视化
• **多页面类型识别**：自动识别 API 文档、GitHub 仓库、YouTube 视频、PDF 文档等，提供专属快捷操作
• **多模态支持**：可以分析页面图片、提取 YouTube 字幕总结视频
• **自定义技能**：创建专属 AI 技能模板，一键调用常用 prompt
• **间隔复习**：基于遗忘曲线的智能复习系统，帮助巩固学习成果
• **书签图谱**：AI 驱动的书签管理，力导向图可视化知识关联

**💡 特色亮点**

• 🔒 数据完全本地化 — 知识库存储在浏览器本地 IndexedDB，不上传任何服务器
• 🤖 多 AI 支持 — 兼容 Claude、OpenAI、DeepSeek、Ollama 等主流 AI API
• 🎯 6 种页面感知 — 自动识别技术文档、GitHub 仓库、API 文档、YouTube、PDF 等
• 📊 学习路径 — AI 基于你的知识库生成个性化学习路线
• 🕸️ 知识图谱 — 力导向图可视化展示知识关联
• ⌨️ 快捷键 — Ctrl+Shift+Y/S/X 快速操作
• 🌙 暗色主题 — 支持跟随系统自动切换

**📖 支持的页面类型**
• 通用网页 — Reader Mode 智能提取
• API 文档 — 自动提取端点列表
• GitHub 仓库 — 分析 README 和目录结构
• YouTube 视频 — 字幕提取与总结
• PDF 文档 — 内容提取与分析

**⚙️ 配置简单**
安装后在设置页面配置 AI API 密钥即可使用。支持多个配置 Profile，一键切换不同 AI 服务。

**🔒 隐私承诺**
• 不收集任何用户数据
• 不集成广告、分析或追踪工具
• 所有数据存储在浏览器本地
• 详细隐私政策：https://github.com/pagewise/pagewise

### English Version

**PageWise** — Your AI-Powered Web Reading Assistant

Browsing technical docs, API references, or GitHub repos? Select any text on the page and ask AI instantly! All answers are automatically organized into a structured knowledge base for later review.

**🚀 Core Features**

• **Smart Q&A**: Select text on any webpage to ask AI — supports streaming output and multi-turn conversations
• **Knowledge Base**: Answers auto-save to a local knowledge base with full-text search, tag filtering, and knowledge graph visualization
• **Multi-Page Intelligence**: Auto-detects API docs, GitHub repos, YouTube videos, PDFs — with tailored quick actions for each
• **Multimodal**: Analyze page images, extract YouTube subtitles and summarize videos
• **Custom Skills**: Create your own AI skill templates for one-click prompt execution
• **Spaced Repetition**: Smart review system based on forgetting curves to reinforce learning
• **Bookmark Graph**: AI-powered bookmark management with force-directed graph visualization

**💡 Highlights**

• 🔒 Fully Local Data — Knowledge base stored in browser's local IndexedDB, never uploaded to any server
• 🤖 Multi-AI Support — Compatible with Claude, OpenAI, DeepSeek, Ollama and other AI APIs
• 🎯 6 Page Type Detection — Auto-identifies tech docs, GitHub repos, API docs, YouTube, PDFs
• 📊 Learning Paths — AI generates personalized learning routes based on your knowledge base
• 🕸️ Knowledge Graph — Force-directed graph visualization of knowledge connections
• ⌨️ Shortcuts — Ctrl+Shift+Y/S/X for quick actions
• 🌙 Dark Theme — Supports system auto-switching

**📖 Supported Page Types**
• Web Pages — Smart Reader Mode extraction
• API Docs — Auto-extract endpoint lists
• GitHub Repos — Analyze README and directory structure
• YouTube Videos — Subtitle extraction & summarization
• PDF Documents — Content extraction & analysis

**⚙️ Easy Setup**
Install the extension and configure your AI API key in the settings page. Supports multiple config profiles for quick switching between AI services.

**🔒 Privacy Promise**
• No user data collected
• No ads, analytics, or trackers
• All data stored locally in the browser
• Privacy policy: https://github.com/pagewise/pagewise

## 权限说明（提交时需说明）

提交 Chrome Web Store 时需说明每个权限的用途：

| 权限 | 审核说明 |
|------|----------|
| `storage` | 存储用户设置和知识库数据到浏览器本地 |
| `sidePanel` | 显示扩展的侧边栏问答界面 |
| `contextMenus` | 提供右键菜单快捷提问功能 |
| `tabs` | 获取当前页面标题和 URL 用于知识归档 |
| `activeTab` | 用户主动操作时临时访问页面内容 |
| `bookmarks` | 读取 Chrome 书签用于书签图谱可视化 |
| `<all_urls>` (content_scripts) | 在任意页面注入内容脚本，支持选中文字提问和页面内容提取 |
| `localhost` / `127.0.0.1` | 连接用户本地运行的 AI 服务（如 Ollama） |
