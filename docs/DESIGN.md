## 核心模块

### ai-client.js — AI 客户端
- 统一封装 Anthropic / OpenAI 兼容 API
- 流式输出支持
- Token 计数和限制

### knowledge-base.js — 知识库
- IndexedDB 存储
- CRUD 操作
- 全文搜索

### skill-engine.js — 技能引擎
- 技能注册与发现
- 页面类型匹配
- 技能执行管道

### memory.js — 记忆系统
- 用户画像管理
- 知识召回（加权检索）
- 偏好学习

### evolution.js — 自进化
- 隐式反馈收集
- 策略自动调优
- 模式学习
