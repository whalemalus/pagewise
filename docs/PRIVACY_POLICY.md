# 隐私政策 — 智阅 PageWise

**最后更新日期：2026 年 5 月 13 日**

## 概述

智阅 PageWise（以下简称"本扩展"）是一款 Chrome 浏览器扩展（Manifest V3），帮助用户在浏览技术网页时即时向 AI 提问，并将回答自动整理成结构化知识库。

**我们高度重视用户隐私。本扩展不收集、不存储、不传输任何用户数据至开发者控制的服务器。**

---

## 1. 数据收集

**本扩展不收集任何用户数据。**

我们不会收集、存储或传输以下信息：

- 个人信息（姓名、邮箱、地址等）
- 浏览历史记录
- Cookie 或追踪标识符
- 设备信息或地理位置数据
- 使用行为分析数据
- 崩溃报告或诊断数据

本扩展不集成任何广告 SDK、分析工具或第三方追踪器。

---

## 2. 数据存储

**所有数据均存储在用户本地浏览器中，使用浏览器原生存储 API。**

| 存储类型 | 用途 | 持久性 |
|---------|------|--------|
| `chrome.storage.sync` | 用户设置、API 配置 Profile | 跟随 Google 账号同步 |
| `chrome.storage.local` | 高亮标注、使用统计、自定义模板、快捷键配置 | 本地持久化 |
| `chrome.storage.session` | 当前对话上下文、临时状态 | 关闭浏览器后自动清除 |
| `IndexedDB` | 知识库条目、对话历史、Wiki 页面、自定义技能、书签索引、学习路径 | 本地持久化 |

**重要说明：**
- API 密钥存储在 `chrome.storage.sync` 中，由 Chrome 浏览器加密保护
- 所有知识库数据存储在浏览器本地 IndexedDB 中
- 任何数据都不会上传到本扩展开发者的服务器

---

## 3. 数据传输

**本扩展仅在用户主动操作时，向用户自行配置的 AI API 端点发送请求。**

### 3.1 AI API 请求

当用户主动提问时，本扩展会将页面内容和用户问题发送到用户在设置中配置的 AI API 地址。支持的 API 端点包括：

| 服务 | 域名 | 说明 |
|------|------|------|
| Anthropic Claude | `api.anthropic.com` | Claude 模型 API |
| OpenAI | `api.openai.com` | GPT 模型 API |
| DeepSeek | `api.deepseek.com` | DeepSeek 模型 API |
| 本地 AI 服务 | `localhost` / `127.0.0.1` | Ollama 等本地模型 |
| 自定义端点 | 用户自行配置 | 任意 OpenAI 兼容 API |

### 3.2 YouTube 字幕提取

当用户浏览 YouTube 视频页面时，本扩展可提取视频字幕用于辅助理解。此功能仅在用户主动触发时执行，请求目标为 YouTube 官方字幕服务。

### 3.3 数据传输控制

- 所有 API 请求均由用户主动触发（点击发送按钮或快捷键）
- API 密钥和端点由用户自行配置和管理
- 用户可随时在设置中修改或删除 API 配置
- 本扩展不会将数据发送到扩展开发者控制的任何服务器

## 4. 权限用途说明

| 权限 | 用途 | 使用场景 |
|------|------|----------|
| `storage` | 本地数据持久化 | 存储用户设置、API 配置、使用统计、自定义模板 |
| `sidePanel` | 显示侧边栏 UI | 提供问答界面、知识库、Wiki、技能面板等核心功能 |
| `contextMenus` | 右键菜单集成 | 提供「用智阅提问」右键快捷操作 |
| `tabs` | 标签页信息查询 | 获取当前页面标题和 URL，用于知识归档和多标签分析 |
| `activeTab` | 当前标签页访问 | 仅在用户主动操作时临时访问当前标签页内容 |
| `bookmarks` | 书签读取 | 读取 Chrome 书签数据用于书签图谱可视化和智能分类 |

### Host 权限

| 域名 | 用途 |
|------|------|
| `api.anthropic.com` | Claude API 通信 |
| `api.openai.com` | OpenAI API 通信 |
| `api.deepseek.com` | DeepSeek API 通信 |
| `localhost` / `127.0.0.1` | 本地 AI 服务通信（如 Ollama） |

---

## 5. 第三方服务

本扩展依赖以下第三方服务，**均由用户自行选择和配置**：

### 5.1 AI API 服务
- 用户自行选择的 AI 模型提供商
- 用于处理用户提问并返回回答
- 数据传输取决于用户选择的服务提供商的隐私政策

### 5.2 浏览器内置 API
- Chrome 浏览器提供的存储、标签页、书签等原生 API
- 遵循 Google Chrome 隐私政策

### 5.3 无第三方 SDK
本扩展不集成以下任何类型的第三方组件：
- ❌ 广告网络
- ❌ 分析/统计工具
- ❌ 用户行为追踪器
- ❌ 社交媒体 SDK
- ❌ 错误追踪服务

---

## 6. Content Security Policy

本扩展声明了严格的 Content Security Policy：

```
script-src 'self'; object-src 'self';
```

- 仅允许加载扩展自身的脚本
- 禁止内联脚本执行
- 禁止 `eval()` 动态代码执行
- 禁止加载外部脚本资源

---

## 7. 数据删除

用户可以随时通过以下方式删除所有本地数据：

1. **扩展内删除**：在设置页面清除知识库和对话历史
2. **批量清除**：使用「清除所有数据」按钮删除所有扩展数据
3. **浏览器设置**：在 Chrome 设置中清除扩展数据
4. **卸载扩展**：卸载后所有本地存储数据将被清除

---

## 8. 儿童隐私

本扩展不针对 13 岁以下儿童，也不会故意收集儿童的个人信息。
