模块: lib/memory.js — export class MemorySystem

API:
- constructor() — 内部创建 KnowledgeBase 实例
- async init() — 初始化 KnowledgeBase + 加载用户画像
- async loadUserProfile() — 从 chrome.storage.sync 加载用户画像
- async saveUserProfile() — 保存用户画像到 chrome.storage.sync
- async learnFromInteraction(question, answer, pageContext) — 从交互中学习用户画像
  - pageContext: { codeBlocks: [{lang}], url: string }
  - 学习编程语言（从 codeBlocks.lang）
  - 学习领域（从 url 提取域名）
  - domains 最多保留 20 个
- async recall(query, aiClient=null) — 三层检索：关键词→评分→AI重排
  - 返回 Array<{type, content, source?, title?, tags?}>
  - type: 'knowledge' | 'user-profile'
- async extractKeywords(query) — 提取关键词，过滤停用词
- async keywordSearch(keywords) — 关键词搜索，匹配 title/summary/question/answer/tags
- scoreRelevance(entry, keywords, query) — 计算相关性分数（标题5分/标签4分/问题3分/摘要2分/回答1分）
- async aiRerank(query, candidates, aiClient) — AI 重排序，返回 top5
- async autoSaveIfWorth(question, answer, pageContext, aiClient) — 自动保存判断
  - 规则1: answer.length >= 100
  - 规则2: 包含技术关键词
  - 返回 null 或 saveEntry 结果
- async getAllEntries(limit) — 透传到 KnowledgeBase
- async getEntry(id) — 透传到 KnowledgeBase
- async deleteEntry(id) — 透传到 KnowledgeBase
- async getAllTags() — 透传到 KnowledgeBase
- async exportMarkdown() — 透传到 KnowledgeBase
- async exportJSON() — 透传到 KnowledgeBase
- extractDomain(url) — 从 URL 提取域名
- async toPrompt(query, aiClient=null) — 生成给 AI 的记忆上下文
