# 隐私政策 — 智阅 PageWise

**最后更新日期：2026 年 4 月 28 日**

## 概述

智阅 PageWise（以下简称"本扩展"）是一款 Chrome 浏览器扩展，帮助用户在浏览技术网页时向 AI 提问，并将回答整理成知识库。我们高度重视用户隐私，本政策说明了本扩展如何处理数据。

## 数据收集

**本扩展不收集任何用户数据。**

我们不会收集、存储或传输以下信息：
- 个人信息（姓名、邮箱、地址等）
- 浏览历史记录
- Cookie 或追踪标识符
- 设备信息或地理位置数据

## 数据存储

**所有数据均存储在用户本地浏览器中。**

本扩展使用以下浏览器本地存储机制：

| 存储类型 | 用途 | 持久性 |
|---------|------|--------|
| `chrome.storage.sync` | 用户设置、API 配置 | 跟随 Google 账号同步 |
| `chrome.storage.local` | 高亮标注、使用统计、自定义模板 | 本地持久化 |
| `chrome.storage.session` | 当前对话上下文 | 关闭浏览器后自动清除 |
| `IndexedDB` | 知识库条目、对话历史、自定义技能 | 本地持久化 |

所有数据完全存储在用户浏览器中，不会上传到任何服务器。

## 数据传输

**本扩展仅向用户自行配置的 AI API 端点发送请求。**

当用户主动提问时，本扩展会将页面内容和用户问题发送到用户在设置中配置的 AI API 地址。可能的 API 端点包括：

- Anthropic Claude API（`api.anthropic.com`）
- OpenAI API（`api.openai.com`）
- DeepSeek API（`api.deepseek.com`）
- 本地 AI 服务（`localhost` / `127.0.0.1`）
- 用户自定义的 API 地址

**重要提示：** 所有 API 请求均由用户主动触发，API 密钥和端点由用户自行配置和管理。本扩展不会将数据发送到扩展开发者控制的任何服务器。

## 第三方服务

本扩展依赖以下第三方服务（均由用户自行选择和配置）：

- **AI API 服务**：用户自行选择的 AI 模型提供商，用于处理用户提问
- **Chrome 浏览器内置 API**：用于本地数据存储和扩展功能

本扩展不集成任何广告、分析、追踪或第三方 SDK。

## 权限使用说明

| 权限 | 用途 |
|------|------|
| `storage` | 存储用户设置和本地数据 |
| `sidePanel` | 显示侧边栏界面 |
| `contextMenus` | 提供右键菜单「用智阅提问」功能 |
| `tabs` | 查询当前标签页信息、与内容脚本通信 |

## 数据删除

用户可以随时通过以下方式删除所有本地数据：
- 在扩展设置页面清除知识库和对话历史
- 在 Chrome 设置中清除扩展数据
- 卸载扩展

## 儿童隐私

本扩展不针对 13 岁以下儿童，也不会故意收集儿童的个人信息。

## 隐私政策变更

如本隐私政策发生重大变更，我们将在扩展更新日志中通知用户。
